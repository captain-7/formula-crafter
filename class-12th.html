Matrices



1. Square Matrix: An m × n matrix will be known as a square matrix when m = n.
2. Diagonal Matrix: A = [aij]m × m will be known as a diagonal matrix if aij = 0, when i ≠ j.
3. Scalar Matrix: A = [aij]n × n is a scalar matrix if aij = 0, when i ≠ j, aij = k, (where k is some constant); and i = j.
4. Identity Matrix: A = [aij]n × n is an identity matrix, if aij = 1, when i = j and aij = 0, when i ≠ j.
5. Zero Matrix: A zero matrix will contain all its elements as zero.
6. Column Matrix: A matrix which is of the form [A]n×1 is called the column matrix.
7. Row Matrix: A matrix that is of the form [A]1×n is called the row matrix.
8. Addition of Matrix: If A = [aij]m×n and B = [yij]m×n, then A + B = [aij +bij]m×n, 1 ≤ i ≤ m, 1 ≤ j ≤ n
9. Subtraction of Matrix: If A = [aij]m×n and B = [bij]m×n, then A – B = [aij – bij]m×n, 1 ≤ i ≤ m, 1 ≤ j ≤ n
10. Multiplication of a matrix by scalar number: Let A = [aij]m×n be a matrix and k is scalar, then kA is another matrix obtained by multiplying each element of A by the scalar k, i.e. if A = [aij]m×n, then kA = [kaij]m×
11. Multiplication of Matrices: Let A and B be two matrices. Then, their product AB is defined, if the number of columns in matrix A is equal to the number of rows in matrix B.
